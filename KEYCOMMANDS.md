# VS Code Key Commands

# Windows Style Key Commands

- <b>PageUp</b> scroll up one page.
- <b>PageDown</b> scroll down one page.
- <b>Ctrl-←</b> skip to the left.
- <b>Ctrl-→</b> skip to the right.
- <b>Home</b> skip to the left all the way to the beginning of the line.
- <b>End</b> skip to the right all the way to the end of the line.


- <b>Ctrl-Shift-←</b> select text to the left.
- <b>Ctrl-Shift-→</b> select text to the right.
- <b>Ctrl-Shift-Home</b> select text to the left all the way to the beginning of the line.
- <b>Ctrl-Shift-End</b> select text to the right all the way to the end of the line.

# Multiple Cursors

- <b>Alt Click</b> multiple line editor with gap selection.
- <b>Alt Shift Click</b> drag multiple line editor.

# Collapse Expand Section

- <b>Ctrl-Shift-[</b> collapse section
- <b>Ctrl-Shift-]</b> expand section

- <b>Ctrl-l</b> select a line
- <b>Ctrl-d</b> select a word

Ctrl-/ comment out the selection
   Does a <!-- --> on Markdown / HTML, and a // on Java or context sensitive to the Language setting
